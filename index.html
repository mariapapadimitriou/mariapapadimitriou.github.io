Projects

Predicting the NBA Playoffs

Developed a logistic regression prediction model using Python scikitlearn libraries to classify the outcome of a matchup of two teams in the NBA by using regular season data to train the model. This model was then used in a playoff simulator where 16 teams and their seed rankings from the season were given as input, and were continuously matched up when predicted the winner each round, resulting in one final winning team as output. The overall playoff simulator was repeated using Monte Carlo simulation and the frequency of each team winning was displayed on a histogram.

Coin Counter Image Recognition Project

Built a database of 1000 images of Canadian coins (including nickels, dimes, quarters, loonies, and toonies), and created a convolutional neural network (CNN) artificial intelligence model using Python PyTorch libraries to identify and classify an image of a coin. An additional program was developed using OpenCV and Hough Circles libraries to identify and crop singular coins from an image of groupings of coins. Each coin was then classified using the CNN, and their monetary value was added to the total balance.

Medicine Tracker Web Application

Built the front and back-end of a Java based web application containing client data on vaccine/dose information to help track next dosage times. A MS Access relational database was used to hold client username/login information, associated drugs used by each client, and specific dosage requirements for each drug. SQL queries were run to pull and display data from the user to the front-end interface. The system performed a total of 3 use cases including (1) adding a medicine to their account, (2) recording when they take the vaccine (where the next time and quantity remaining is displayed), and (3) recording when they take their required medication.
